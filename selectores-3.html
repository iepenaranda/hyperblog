Pseudoclases: Permiten reconocer las acciones que hace el usuario, como clickear o pasar el puntero por encima de un elemento

  • :active
  • :focus
  • :hover
  • :nth-child(n)

Pseudoelementos: Permiten acceder a elementos html como la primera letra, el texto de un campo.

  • ::after
  • ::before
  • ::first-letter
  • ::placeholder
